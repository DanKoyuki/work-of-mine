Pokemon Unite

Pokemon Unite adalah game MOBA terbaru yang release pada bulan September kemarin. Bebeda dengan MOBA lainnya dimana untuk memenangkan match pemain diharuskan untuk menghancurkan markas musuh, Pokemon Unite memiliki gaya permainan yang baru dimana pemain harus mencetak score sebanyak mungkin dalam waktu sepuluh menit untuk dapat memenangkan match. Dengan adanya batas waktu membuat saya dapat mengatur waktu untuk bermain game menjadi lebih baik lagi. Pada game ini saya juga telah mencapai rank tertingginya yaitu Master Rank.
